 extemp
=============

Gentoo [srpy](https://github.com/drdbrr/srpy) ebuild repository.

Usage
-----

Add this gentoo repository to system:
* Remote URL:
```bash
eselect repository add extemp git https://github.com/drdbrr/extemp.git && emaint sync -r extemp
```
